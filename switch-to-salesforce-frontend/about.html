About Switch To Salesforce

A documentation-style blog for people moving into Salesforce roles and leveling up on the platform.

Switch To Salesforce teaches Salesforce development and administration through clear, long-form articles — inspired by modern docs sites and product blogs. Expect tutorials on Apex, Lightning Web Components, Flow Builder, data modeling, security, interviews, certification paths, and career roadmaps.

Who it is for

  • Developers transitioning to Salesforce
  • Salesforce administrators expanding their skills
  • Students preparing for certifications
  • Professionals preparing for Salesforce interviews

How it is built

  • Frontend — static HTML, CSS, and JavaScript; deployable to Vercel, Netlify, or Cloudflare Pages.
  • Backend — separate Express + MongoDB API with JWT admin auth and image uploads. The browser only talks to REST endpoints.

Features

Responsive layout, dark mode, reading progress on articles, syntax highlighting, and copy-to-clipboard for code blocks.
